
Title of the project is called Tic Tac Toe.
Tic tac toe is a game developed using the following languages,
1:HTML(Hyper Text Markup Language)
2:CSS(Cascading Style Sheet)
3:Javascript.
Tic Tac Toe game can be played either Solo vs Computer or Multiplayer againt another Player.
This projects works on various screen sizes such as mobile, tablet, laptops as well as Desktops.
This projects has various hovering states for all interactive elements on the page.
